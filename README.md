# Content Verification Service

AI-powered content verification service for business platforms. Handles both text and image content verification.

## Features

- **Text Classification**: NLP-based text content classification
- **Image Classification**: CNN-based image content classification
- **Multi-modal Fusion**: Combines text and image predictions
- **Business Rules**: Applies business-specific verification rules
- **Restricted Categories**: Blocks prohibited content categories
- **REST API**: FastAPI-based RESTful API endpoints
- **Real-time Verification**: Instant content verification
